# Frontend API Client Rules

## Generated Client

The API client at `src/lib/api/client.ts` is auto-generated by Kiota from the backend OpenAPI spec.

- **Never** edit `client.ts` manually
- Run `npm run generate:client` to regenerate after backend API changes
- Run `npm run generate:watch` during development to auto-regenerate on spec changes

## Custom Hooks

All API calls must be wrapped in React Query hooks in `src/lib/hooks/`:

```typescript
// Existing hooks:
// useApi.ts     -- generic API wrapper
// useAuth.ts    -- authentication (login, register, profile)
// useImages.ts  -- image/media data fetching
```

- Always use React Query for server state (caching, refetching, loading states)
- Never store API response data in Zustand stores
- Use the generated API client types for type safety
